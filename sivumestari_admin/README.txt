Sivumestari Admin
==============
This is the administrative heart of Sivumestari. It provides integration with popular UX/UI
enhancement modules and exposes a variety of end user settings for Sivumestari.

Key Features
* Customized Content Editing Experience
* Custom Style for the Media Browser
* Defines an "Administor" and "Editor user role
* Creates Sivumestari Administrative Section (admin/sivumestari) for Sivumestari Configuration
* Provides Pane Library, Layout Library, and Page Library Administrative Pages
